# AWS API Gateway Lambda Example

This is a sample application that setups lambda and api gateway and can be deployed using AWS CDK 

## Technologies
- .net core
- Lambda
- API Gateway
- AWS CDK C#
- Github Actions

## Methods
- /{name} - Does an upper on string
- POST - Combines the first and last name
